**App de controle de estoque — Frutal, MG**

---

## Como instalar no celular (passo a passo)

### 1. Criar conta gratuita no GitHub
Acesse https://github.com e crie uma conta grátis.

### 2. Criar um repositório
- Clique em "New repository"
- Nome: `jb-estoque`
- Marque "Public"
- Clique em "Create repository"

### 3. Fazer upload dos arquivos
Arraste todos estes arquivos para o repositório:
- `index.html`
- `manifest.json`
- `sw.js`
- pasta `icons/` com os dois ícones

### 4. Ativar o GitHub Pages
- Vá em Settings → Pages
- Em "Source", selecione: **Deploy from a branch**
- Branch: **main** / pasta: **/ (root)**
- Clique em Save

### 5. Acessar o app
Após ~2 minutos, seu app estará em:
`https://SEU_USUARIO.github.io/jb-estoque`

### 6. Instalar no celular
**Android:**
- Abra o link no Chrome
- Toque no menu (⋮) → "Adicionar à tela inicial"
- Confirme → ícone aparece na tela!

**iPhone:**
- Abra o link no Safari
- Toque no botão compartilhar (⬆)
- "Adicionar à Tela de Início"
- Confirme → ícone aparece na tela!

## Funcionalidades
- ✅ Controle de estoque (ingredientes, embalagens, limpeza, congelados)
- ✅ Alertas automáticos de estoque baixo e crítico
- ✅ Registro de entradas e saídas com histórico
- ✅ Ajuste rápido de quantidade (+/-)
- ✅ Exportar CSV para planilha
- ✅ Funciona offline (sem internet)
- ✅ Dados salvos no celular
